# Plan: Write README.md for ark-agent-cli

## Overview

Create a comprehensive README.md for the ARK Agent CLI - an interactive terminal UI for exploring biomedical knowledge graphs using AI agents. This is the CLI interface for the ARK (Adaptive Retriever of Knowledge) research project.

## Key Information Gathered

### Project Identity
- **Name**: ark-agent-cli (branded as "GraphAgent" in the UI)
- **Purpose**: Interactive TUI for querying biomedical knowledge graphs with AI
- **Organization**: MIMS Harvard (Zitnik Lab)
- **Related**: ARK research paper (arXiv:2601.13969), main ARK repo

### Technology Stack
- Runtime: Bun
- Language: TypeScript
- UI: React 19 + @ai-tui/core + @opentui/react
- AI: Vercel AI SDK + Anthropic (Claude Opus 4.5)
- Database: PostgreSQL + Drizzle ORM
- Package Manager: pnpm

### Three Knowledge Graph Agents
1. **PrimeKG** - Precision medicine-oriented KG
2. **AfriMedKG** - Pan-African medical Q&A
3. **OptimusKG** - Multimodal precision medicine KG

### Available Tools
- `listAvailableGraphs` - List all knowledge graphs
- `findNodesByName` - Search nodes by name
- `getNodeDetails` - Get node details
- `getNeighborsByNodeId` - Get node neighbors
- `searchInSurroundings` - Search within 1-2 hops
- `findPaths` - Find paths between nodes
